## Installation & Setup

<AccordionGroup>
  <Accordion title="How do I install Elkar in Microsoft Excel?">
    You have two installation options:

    ### Option 1: Direct from Microsoft AppSource (Recommended)

    1. Visit [Microsoft AppSource](https://appsource.microsoft.com/en-us/product/office/WA200008734?tab=Overview)
    2. Click **"Get it now"**
    3. Sign in with your Microsoft account if prompted
    4. Follow the installation instructions

    ### Option 2: Install from within Excel

    1. **Open Excel** and go to the **Insert** tab
    2. Click **"Get Add-ins"** (or "Office Add-ins")
    3. **Search for "Elkar"** in the store
    4. Click **"Add"** to install
    5. Once installed, go to **Home** tab â†’ **Elkar** â†’ Sign in and open Chat

    <Tip>

## Data & Privacy

<AccordionGroup>
  <Accordion title="How does Elkar keep my data safe?">
    We take data privacy seriously and implement multiple security layers:

    - **LLM-agnostic**: Choose your preferred model provider (OpenAI, Claude, Mistral, Gemini)
    - **No data retention**: Your data is automatically deleted after processing
    - **Encryption in transit**: All data is encrypted using TLS 1.2 or higher
    - **Partnership guarantees**: Our LLM partnerships ensure your data is never used for training
    - **ISO27001 certification**: Currently undergoing certification process

    For more details, see our [Data Security page](/help/data-security).
  </Accordion>
  <Accordion title="What happens when I enable chat history?">
    When chat history is enabled:

    - Your conversation history is **encrypted and securely stored** on Elkar's backend
    - You can **access previous conversations** across sessions
    - You can **disable this feature** anytime from your [dashboard settings](https://sheets-app.elkar.co/settings)
    - You can **delete past conversations** individually whenever you want

    This feature is optional and can be turned off at any time.
  </Accordion>
  <Accordion title="Are all requests routed through Elkar?">
    Yes, even if you use your own API key, requests are routed through Elkar's backend. This is where we:

    - Build optimized prompts for better results
    - Coordinate agent-based task execution
    - Ensure consistent, high-quality outputs
    - Route intelligently across different models

    Think of it as a conductor orchestrating your AI workflow for the best possible experience.
  </Accordion>
</AccordionGroup>
